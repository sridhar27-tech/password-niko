# password-niko

A premium Password Generator with a real-time Strength Indicator built using the Web Crypto API.

## Features
- **Secure Randomization**: Uses `window.crypto.getRandomValues`.
- **Strength Analysis**: Real-time evaluation of password complexity.
- **Manual Input**: Type your own passwords to check their strength.
- **Sleek UI**: Glassmorphism design with dark mode.
- **Copy to Clipboard**: Instant feedback when copying.

## Technology Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla ES6+)
- Web Crypto API

## How to Run
Simply open `index.html` in any modern web browser.
